MiddleMan入門
======================

# 目的 #
MiddleManを使ってサイトの作成・更新が出来るようになる。

# 前提 #
| ソフトウェア   | バージョン   | 備考        |
|:---------------|:-------------|:------------|
| OS X           |10.8.5        |             |
| ruby           |1.9.3p392     |             |


# 構成 #
+ インストール


# 詳細 #

## インストール ##

    $ rvm gemset create middle_man
    $ rvm use ruby-1.9.3-p392@middle_man
    $ gem install middleman

## 新しいサイトの開発を始める ##
